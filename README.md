# Projeto de Exemplo

Este é um projeto de exemplo para mostrar aos alunos como trabalhar com GIT

A ideia central deste projeto de exemplo é mostrar os seguintes comandos do GIT:
- `git add`
- `git commit -m`
- `git pull`
- `git push`
- `git status`
- `git restore --staged <arquivo>`
- `git log`
- `git log --oneline`
